FMCSA Data Visualizations

Overview

Last summer (June 2013 - August 2013), I worked for Macrosys Research & Technology as part of a team working on a DOT contract in Cambridge, MA. The project that I worked on for most of the summer was a set of dynamic, interactive data visualizations. The data to be visualized was records of motor carrier safety violations, and would be embedded in the new website Macrosys had been tasked with building for the FMCSA (Federal Motor Carrier Safety Administration).

Tools

  • HTML
  • jQuery
  • d3.js

This project used only javascript and HTML. All the data was embedded in the HTML, and I used jQuery for easy extraction of that data. The lion's share of the code, however, used Data-Driven Documents, or d3.js, a powerful data visualization library written by Mike Bostock. In short, d3 allows you to bind sets of data to the DOM, making it easy to leverage CSS, SVG, and HTML to create dynamic, and, as its name suggests, data-driven, visualizations.
